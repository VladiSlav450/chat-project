Sample page for Thalassa templates

This page is generated by Thalassa CMS using one of the templates which come along with the CMS as an example. The source file for this page is 

  $site_source/db/pages/sample_page

(where $site_source is the root directory for your site source, initially created as a copy of the template of choice). The source file contains the following header: 

  id: sample_page
  type: page
  title: Sample page for Thalassa templates
  format: texbreaks

Please note that no date is displayed at the page, it's where the page type differs both from story and feed.

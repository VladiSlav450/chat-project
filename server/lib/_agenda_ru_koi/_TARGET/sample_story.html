Sample story page for Thalassa templates

Mon Jul 17 13:12:45 2023 UTC

This story page is generated by Thalassa CMS using one of the templates which come along with the CMS as an example. Comments are enabled on this page to demonstrate the possibility. The source file for this page is 

  $site_source/db/pages/sample_story

(where $site_source is the root directory for your site source, initially created as a copy of the template of choice). The source file contains the following header: 

  id: sample_story
  unixtime: 1689599565
  type: story
  title: Sample page for Thalassa templates
  format: texbreaks
  comments: enabled

Please note the date displayed at the top of this page. This is where the story type differs from the default (page).
